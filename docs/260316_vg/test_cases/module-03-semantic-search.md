# 纯语义检索

**覆盖目标**：query embedding、KNN、路径过滤、阈值、top_k、空结果、语义相关性。

## 前置条件

- 已完成至少一次成功建索引
- 存在具备语义相关性的夹具（含中文文档）
- 模型缓存可读

## 约束

- 语义结果的合理性带主观性，需人工确认样例集
- score 为 `1 / (1 + distance)`，断言时用相对排序而非固定绝对值
